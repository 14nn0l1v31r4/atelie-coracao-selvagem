# Ateliê Coração Selvagem

Landing page demonstrativa e institucional para o Ateliê Coração Selvagem, criado com Vite, React, TypeScript e Tailwind CSS.

## Recursos

- Layout responsivo para celular, tablet e computador.
- Dois temas: **Luz natural** e **Verde profundo**.
- Preferência de tema salva no navegador.
- Galeria de peças botânicas e outras criações.
- Seção sobre o processo artesanal.
- Links diretos para o Instagram do ateliê.
- Sem recursos comerciais, preços, carrinho ou catálogo de vendas.

## Executar localmente

```bash
npm install
npm run dev
```

## Gerar versão de produção

```bash
npm run build
```

Os arquivos finais serão criados na pasta `dist/`.

## Ajustes responsivos da versão 1.1

- Cabeçalho móvel compacto, com Instagram dentro do menu e sem extrapolar a largura da tela.
- Tipografia principal limitada em monitores grandes para preservar a proporção editorial.
- Escalas específicas para celulares de 320 px a 767 px.
- Proteções contra rolagem horizontal acidental em todo o documento.
